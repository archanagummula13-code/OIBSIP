🎯 Number Guessing Game (Java)
📌 Description
The Number Guessing Game is a simple Java console-based application where the computer generates a random number and the player has to guess it. The program provides hints such as "Too High" or "Too Low" to help the player find the correct number within limited attempts. This project helps beginners understand basic programming concepts in Java.
✨ Features
    Random number generation
    User input from console
    Hints to guide the player (Too High / Too Low)
    Limited number of attempts
    Simple and interactive gameplay
🛠 Technologies Used
    Java
    VS Code / Any Java IDE
▶ How to Run
    Clone the repository
    Open the project in VS Code or any Java IDE
    Compile the Java file
    Run the program
    Start guessing the number!
🎓 Learning Outcomes
 This project helps in understanding:
    Loops
    Conditional statements
    Random number generation
    User input handling
